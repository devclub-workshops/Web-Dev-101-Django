CSS Syntax

CSS can alter how an element is rendered on a browser.
You can alter the elements in 3 different ways.

Direct Elements

If you alter an element directly, every similar element will be rendered in the same way.
To do that, you write the element name, eg. h1 followed by curly braces {}

Class

When you want to modify a particular unique element on the page, you can use class to avoid styling all other similar elements.
You can have multiple elements having same class. Classes can make styling of a webpage modular.
