Methods To Use CSS

CSS decides how a HTML element is rendered on your screen. You can link CSS Rules to HTML page in three different manners.
The most common way is to use external CSS that allows the project source code to remain clean and allow multiple people to collaborate.

Inline CSS

When you apply style attribute directly to a HTML element, it's known as inline css.
The stlye applied is unique to that particular element and won't be applied other similar elements.

When you want to apply style to a single page you can use internal CSS.
To declare internal CSS, you create style tag in the head of the HTML page.

External CSS

An external style sheet is used to define the style for many HTML pages.
With an external style sheet, you can change the look of an entire web site, by changing one file!
To do this you create a file with .css extension and write all your css code in it and link it to the HTML page you want.
